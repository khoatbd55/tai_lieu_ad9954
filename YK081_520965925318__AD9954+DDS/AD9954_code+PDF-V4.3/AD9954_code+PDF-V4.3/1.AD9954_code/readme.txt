【*】简介

-工程名称：AD9954-STM32-DDS驱动板-Vx.x
-实验平台: 康威DDS驱动板
-MDK版本：KEIL 5.14.0
-ST固件库版本：3.5.0
	工程文件位于对应文件夹的"\USER"目录下，用keil5软件打开VirtualCOMPort.uvprojx即可

程序均可直接配合本店DDS驱动板直接使用
AD9954两个通道输出的频率相同，相位相反。功能十分强大，本店提供例程仅为部分功能，
如需例程以外的其他功能需要用户自己修改代码实现

【 ！】实验操作：
1、使用排线连接驱动板与AD9954模块，驱动板上排线方向唯一，驱动板以及DDS模块上，板子背面双排针，方形焊盘为双排针1脚，对应连接即可
2、使用5V电源连接AD9954模块，使用5V电源连接DDS驱动板，
3、AD9954模块侧面输出高频头，使用SMA转BNC屏蔽线，连接示波器与模块
3、编译该代码，使用jlink下载器下载代码到驱动板
4、确保AD9954模块已上电。按下驱动板复位键(或驱动板重新上电)，调整示波器时间档位到合适位置,即可观察到输出波形

5、如使用自己的STM32开发板，请按下方描述接线，如使用其他引脚请自行移植更改程序。
/*********************************************************************************************/

【*】 引脚分配
	
STM32控制板	模块丝印		芯片引脚名	芯片功能
PA3		----->CS		CS		片选
PA4		----->SCK		SCLK		串行时钟
PA5		<----->SDIO		SDIO		串行数据I/O
PC0		----->OSK    		OSK		编程时用于控制成形开关键控功能方向的输入引脚
用于操作。
PA2		----->PS1		PS1 		用于选择内部相位/频率配置文件之一的输入引脚
PB10		----->PS0		PS0 		用于选择内部相位/频率配置文件之一的输入引脚
PA7		----->UPD		I/O UPDATE	上升沿将内部缓冲存储器的内容传输到I/O寄存器。
PA8		<-----SDO		SDO 		串行数据输出
PB1		----->IOSY		IOSYNC 		串行端口控制器的异步主动高复位。
PA6		----->RST 		RESET 		硬件复位引脚
PB0		----->PWR		PWRDWNCTL 	用作外部断电控制

GND		<---->GND		控制板与AD9954模块需共地。
浮空		------其他		所有未说明但模块有留出管脚，未使用可直接浮空，功能请参考数据手册
